html 基础

html 基础

什么是 html

html 是超文本标记语言(标签语言), 超文本指 图片，音频，视频，表单，链接

html 的历史

html 的诞生是 1991 年，html5 的广泛普及是在 2012 年

html 的作用

实现网页上能够看到的整体结构，包括图片，链接，视频，影片等等(网页交互，弹窗，动画)

html5 的基本结构

常用标签

标题标签 h

h 标签一共有六个，分别代表的是不同的标题等级，h1 是最大的标题标签，h6 是最新的标题标签，如果超出 h6 范围，统一是为无效视为无效标签会变成普通文字

注意:

  • 页面存在小字体,经理不要使用 h6 或者其他小字体标签，应该使用 css 属性修改字体(font-size)
  • h1 只存在一个页面(SEO 搜索引擎)

段落标签 p

段落标签会独占一整行，并会自带间隙，一般使用在文章段落

换行标签 br

可以使文本获取标签强制换行，属于单标签元素

水平线标签，分割线标签 hr

在 html 页面创建一条水平线，独占一行，作用适用于分割上下部分内容

文本标签 span

span 适用于文本内容，不会独占一行，会与其他的文本内容共占一行，并且不能设置宽高(行内，内联元素)

图片标签 img

用于展示图片资源的标签(支持网络，本地资源)

常用属性

  • src 图片资源地址，可以写相对路径、绝对路径、网络资源
  • alt 图片加载失败时，显示对应的文本信息(利于 SEO 搜索)
  • srcset 处理页面在不同设备像素密度对图片选择显示
  • width 图片的宽度,(默认 px 单位)
  • heitght 图片的高度,(默认 px 单位)

超链接标签 a

html 可以使用超链接与网络进行文档的链接，点击链接跳转到对应的网络或者是本地地址

超链接标签锚点的使用

利用 id 全局属性(所有标签拥有的属性),创建 html 文档中唯一的书签标记,然后可以通过 a 链接跳转到 id 元素
