数组 Array

数组对象 是使用单独的变量名来存储一系列的值，每个值称为当前数组的成员，第一位值的索引值(下标值)为 0

创建数组

  • new 操作符好 (Array 构造函数，实例化对象)
  • 字面量

访问数组成员

使用[索引值]的方法去访问数组成员，如果访问的成员不存在即返回 undefined，[索引值]也可以做赋值操作(将新的值赋给对应下标的数组成员)

检测数组 (是否为数组)

  • isArray 检测是否为数组(返回布尔值)

数组的方法

  • concat(a1,a2,…)
  • join()
  • reverse()
  • slice(startIndex, endIndex)
  • splice(arg1, arg2, arg3)
  • IndexOf(arg1, arg2)
  • lastIndexOf(arg1, arg2)
  • includes
  • push、pop、shift、unshift

数组的遍历方法

计算机的词汇(循环、遍历、迭代、递归)

  • foreach(cb)
  • map(cb)
  • filter(cb)
  • every(cb)
  • some(cb)
  • reduce(cb,init)
  • sort()
  • fill()
  • find()
  • findIndex()
