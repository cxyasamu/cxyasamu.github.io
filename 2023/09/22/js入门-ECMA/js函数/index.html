函数

函数是完成某个功能的一组语句，定义函数之后，函数体里面的代码可以重复使用(定义函数)

函数的参数

在调用函数的时候，需要传递参数到函数内部，就要使用函数的参数传递的操作，函数内部存在两个隐藏参数(默认)，分别是 this 和 arguments

  • this (函数的执行上下文对象)，以函数形式调用，this 指向的是 window 对象

  • arguments 是一个类数组(无数组的方法、有数组的长度)，可以通过索引/下标值操作数据以及获取长度，length 属性当前实参的个数，callee 属性对应的是函数本身

js 执行环境(执行上下文)

决定上下文定义了变量或者函数有权访问其他的数据，决定了他们各自的行为(js 出现多个执行环境)

  • 什么是作用域?

它指的是变量或函数具有访问权限的代码区域;在 js 中，作用域是在函数中维护的，表示变量或者函数起作用的区域，例如函数中的作用域指代的是在函数执行上下文中、全局作用域指代的是全局的执行上下文

  • 全局作用域

声明在函数外部的变量、函数，在代码中的任何地方都可以访问到

  • 函数作用域(局部作用域)

声明在函数内部的变量、函数，局部作用于固定的代码片段中可以访问(代码片段就是函数执行代码区域)

  • 块级作用域(ES6 提出)

使用 let/const，在{}里面的代码就会产生块级作用域，例如 for、if、while

作用域链

函数内部如果在当前的作用域中查找不到对应的变量/值，自动会向上一层作用域进行查找，直到查找到全局的作用域，如果还找不到就会直接报错 xxx is not defined;这个过程会形成一条查找链条，这个链条就是作用域链
