预处理器

css 并不是一个编程语言(编程语言:判断、循环、变量、函数)，预处理器专门针对 css 研发的一种编程语言，进行 css 的样式设计，最后在编译成 css 文件，常见的预处理器 sass、less

sass

scss 本来就是 sass 的一个升级版，可以看作像 css3 其实就是 css 升级版的意思，scss 是 sass 的第三代语法，scss 可以使用判断、循环、变量、函数等操作，文件的后缀名为 scss(编译之前)，最后编译出去的文件是 css 文件(编译之后)
