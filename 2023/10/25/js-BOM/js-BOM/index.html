BOM

浏览器对象模型 Browser Object Model，浏览器对象模型提供独立的 API，可以和浏览器窗口进行交互，BOM 由多个对象组成，其中 window 对象就是 BOM 的顶层对象，其余的对象都是 window 的子对象

BOM 对象

  • window 对象 (顶层对象)

  • document 对象 (文档对象)

  • location 对象 (浏览器当前的 URL 对象)

  • navigator 对象 (浏览器本身信息)
  • screen 对象 (浏览器屏幕信息)
  • history 对象 (浏览器的访问历史信息)

位置

offset 系列

offset 偏移，包括这个元素在文档中所占用位置大小

5.offsetLeft
当前元素距离(offsetParent)左边的偏移值
公式:
offsetParent 元素的 offsetLeft + 其他中间元素 offsetWidth +offsetLeft

client 系列

指元素本身的可视内容，不包括(border、滚动条、超出内容被隐藏部分)

scroll 系列

指元素滚动，包括这个元素没显示出来的实际宽度和高度，包括 padding，不包括(border、滚动条)
