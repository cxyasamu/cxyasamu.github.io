盒子模型

什么是盒子模型

所有的 html 元素都可以看作是一个盒子，这个盒子包括：内外边距、边框、实际内容，一个完整的盒子模型由 width、height 组成的内容区、border 边框区、padding 内边距区、margin 外边距区

W3C 盒子模型(标准盒模型)

由 width、height 组成的内容区、border 边框区、padding 内边距区、margin 外边距区组成

IE 盒子模型(怪异盒模型)

width 由左右 border、内容宽度、左右 padding，height 由上下 border、上下 padding、内容高度组成

内外边距(padding、margin)

外边距用于定义元素的周围空间(间隙)，内外边距是控制当前元素和其他内部元素之间的距离

边框 border

设置边框颜色、样式、宽度

盒子阴影 box-shadow

文字阴影 text-shadow
