选择器

选择器

元素选择器

通过指定元素名称去设置样式

群组选择器

可以将多个选择器通过’,’连接多个选择器的样式,减少代码的冗余

通配选择器

能够匹配所有元素的选择器(重置浏览器默认样式或者 html 标签自带样式)

类名选择器

别名 class 选择器,通过元素的 class 全局属性去设置样式

ID 选择器

通过元素的 id 全局属性去设置样式,注意一个 ID 选择器只能存在一个元素的 id 属性上(规范),最好不要在 id 选择器上设置样式

选择器之间的权重(优先级)

  • !important 无视优先级
  • 行内样式 优先级 1-0-0-0
  • ID 选择器 优先级 0-1-0-0
  • 类选择器 优先级 0-0-1-0
  • 元素选择器 优先级 0-0-0-1
  • 通配选择器 优先级 0-0-0-0

后代选择器

语法需要使用空格去匹配当前元素内部的选择器

子代选择器

语法需要使用 > 去匹配当前选择器的子代选择器

兄弟选择器

语法需要使用 ~ 去匹配当前同级别的选择器(向下同级)

相邻选择器

语法需要使用 + 去匹配当前选择器相邻的选择器(向下相邻)

伪类选择器

  • 结构类型
  • 用户行为类型

伪元素选择器

在当前元素插入虚拟的元素结构，例如装饰前置和后置。首字母大写操作

  • 装饰首字母 ::first-letter
  • 修饰元素的前置/后置 插入虚拟元素(不存在 html 结构中，只存在 css 中，只能给非置换元素设置)

块级元素和行内元素

  • 块级元素

    • 独占一行，宽度由客户端屏幕大小决定
    • 可设置宽高、内外边距

  • 行内元素

    • 可以和其他行内元素共占一行，宽度由元素内容控制
    • 不可设置宽高，左右外边距有效、内边距有效(置换元素除外)
  • 置换元素

    内容不受 css 视觉控制(css 渲染模型)，默认本身就存在固定的尺寸(宽、高、宽高比)
