自定义列表

表格标签 table

基础语法

  • table 用于定义一个表格区域
  • thead 表格的头部区域
  • tbody 表格主体区域
  • tfoot 表格尾部区域
  • caption 表格标题
  • tr 表格行标签
  • td 表格单元格
  • th 表头的单元格

table 属性

  • boder 表格的边框(默认单位 px)
  • width 表格的宽度
  • height 表格的高度
  • cellspacing 表格单元格之间的间隙
  • cellpadding 表格单元格内边距
