css 基础

中文名:层叠样式表,是属于计算机语言(表现 html 文件),主要作用是装饰 html 标签,还可以负责页面的布局(主要)

css 历史

  • 1994 年，第一个 css 提案的发布(web开始流行)，css 的诞生
  • 2001 年，微软公司发布 IE6 浏览器，该浏览器拥有独特的渲染机制(困扰css的兼容性)
  • 2014 年，微软公司发布 IE10 浏览器，兼容性有所优化(针对兼容性问题，发布了部分的预处理器 — sass、less；后处理器 — postcss)

css 的特点

  • 表现与结构分离

    将 css 样式写道对应的 css 文件中，html 文件单独负责页面结构(结构清晰简洁)

  • 支持多设备样式

    pc、移动、ipad、超大屏

  • 易于维护和改版

    只需要操作 css 文件，那么对应的 html 文件的内容会自动更新

  • 简化设计开发

    css 的样式可以通过继承的方式去使用(层叠效果),不需要写重复的样式

    css 语法的结构

    选择器和声明快，声明块包括对应 css 属性和值，声明块可以拥有多个属性

引入 css 样式

  • 行内/内联样式(不推荐)(权重最高,其他的样式会被直接覆盖,遵循就近原则)

  • 内部/页内样式/嵌入式样式

  • 外部/外链样式(推荐)

css 三大特征

  • 继承性

    子元素会继承父元素的某些样式，例如:文本颜色、大小、文本对齐方式、字体族

  • 层叠性

    相同的选择器给同一个元素设置不同/相同样式的时候,如果是相同样式，后者会覆盖前者的样式，如果存在不同样式，直接层叠/叠加样式

  • 优先级(权重)

    选择器相同，执行层叠性;选择器不同，根据选择器的权重执行
