定位

属于 html 和 css 的布局方式,其他的布局方式:流式布局(百分比布局)、弹性布局(flex)、浮动布局(float)、定位布局在涉及到多个元素层叠的情况下，推荐使用定位布局

绝对定位 absolute

相对于当前元素(不为静态定位)的父元素定位,如果当前元素不存在定位,往上一层元素查找，直到查找到不为静态定位元素位置，最终还找不到，那么默认从屏幕的最左上角进行定位(根元素定位)，可以通过设置 left、top、bottom、right 进行位置调整

注意:不可以和浮动一起使用，float 会失效

相对定位 relative(正常文档流)

相对自身元素所处位置进行定位，可以通过设置 left、top、bottom、right 进行位置调整(类似 margin 属性)，移动后之前的位置会一直保留(占据)

固定定位 fixed(脱离文档流)

相对于浏览器窗口进行定位，可以通过设置 left、top、bottom、right 进行位置调整(默认层级最高)

粘性定位 sticky(未到达粘性值不会脱离文档流)

设置粘性定位的元素默认情况下为相对定位,当滚动到粘性值的时候,会切换为固定的固定定位,直到父元素脱离屏幕(消失)

层叠级别 z-index(定位不为静态定位时使用)

设置定位元素层叠级别，默认 0，值越高层级越高(如果层级相同的情况下,默认后面的元素会覆盖到最上层)
